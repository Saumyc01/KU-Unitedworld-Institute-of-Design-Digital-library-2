Karnavati University Digital Library - PDFs Folder
===================================================

Place your course-wise and e-book PDF files in this folder.

Notes:
- Keep file names consistent with the paths used in:
  - data/ebooks.json
  - data/courses.json
- Example:
  - data/ebooks.json -> "pdfPath": "pdfs/introduction-to-data-structures.pdf"
  - This file should be located at: /pdfs/introduction-to-data-structures.pdf

When deployed to GitHub Pages:
- All paths are relative to the project root.
- Ensure the "pdfs" folder is committed and pushed along with the site.
